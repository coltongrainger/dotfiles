Determine Equation of a Line Given a Graph.

This method is pretty easy if the slope is an integer or simple fraction like \(\frac{2}{3}\) or \(-\frac{1}{2}\text{.}\)

Using the previous graph, which we show again at the right, recall the slope is \(2\) and we can see that the \(y\)-intercept is \(0\text{.}\)

Therefore, the equation for the line is \(y = 2x.\checkmark\)
