How can we graph these two? The second one is easy: it is a line, already in \(y = mx + b\) format. The \(y\)-intercept is \(\frac{1}{2}\) and the slope is \(1\text{.}\) We can graph the first equation by plotting points; or, if you happen to know what the graph of \(y = \sqrt{x}\) looks like, you can stretch the graph vertically to get \(y = 2\sqrt{x}\text{,}\) since all the \(y\)-values will double.
