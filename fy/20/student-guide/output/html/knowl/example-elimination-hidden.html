Solve the following system of equations by elimination.

\begin{align*} 3x + 4y &= 1\\ 2x - y &= 8 \end{align*}

  1. The first question is: how do we get one of these variables to have the same coefficient in both equations? To get the \(x\) coefficients to be the same, we would have to multiply the top equation by \(2\) and the bottom by \(3\text{.}\) It is much easier with \(y\text{;}\) if we simply multiply the bottom equation by \(4\text{,}\) then the two \(y\) values will both be multiplied by \(4\text{:}\)

    \begin{align*} 3x + 4y &= 1\\ 8x - 4y &= 32 \end{align*}

  2. Now we either add or subtract the two equations. In this case, we have \(4y\) on top, and \(-4y\) on the bottom; so if we add them, they will add to zero. (If the bottom had a \(+4y\) we would have to subtract the two equations to eliminate the \(y\)'s.)

    \begin{align*} 3x + 4y &= 1\\ +\underline{(8x - 4y} &\underline{\,= 32)}\\ 11x + 0y &= 33 \end{align*}

  3. Once again, we are left with only one variable: \(11x = 33.\) We can solve this equation to find that \(x = 3\text{.}\)

  4. Plug \(x = 3\) back in to either of the original equations to find \(y = -2\) as before.\(\checkmark\)
