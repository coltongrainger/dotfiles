( Instructor solution preview: show the student solution after due date. )SOLUTION If the function is exponential, its formula is of the form \(y=ab^x\text{.}\) Since (0,6) is on the graph when x = 0, \(y = ab^0 = a = 6\text{.}\) Using the other point at (2,96) we can solve for b: \(y = ab^x = 6(b)^2 = 96\) so \(b^2 = 96/6 = 16\) and \(b = (16)^{1/2} = 4\) The formula is therefore \(y = 6(4)^x\text{.}\)
