Item 1.

The first question is: how do we get one of these variables to have the same coefficient in both equations? To get the \(x\) coefficients to be the same, we would have to multiply the top equation by \(2\) and the bottom by \(3\text{.}\) It is much easier with \(y\text{;}\) if we simply multiply the bottom equation by \(4\text{,}\) then the two \(y\) values will both be multiplied by \(4\text{:}\)

\begin{align*} 3x + 4y &= 1\\ 8x - 4y &= 32 \end{align*}
